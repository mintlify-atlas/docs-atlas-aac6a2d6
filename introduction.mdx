Kubewall is an open-source, single-binary Kubernetes dashboard that provides a simple and rich real-time interface to manage and investigate your clusters. With multi-cluster management and AI integration, Kubewall streamlines Kubernetes operations for DevOps teams of all sizes.

<Note>
Kubewall is currently under active development. We welcome your feedback and contributions.
</Note>

## Why Kubewall?

Kubewall combines simplicity with powerful features to help you manage Kubernetes clusters more effectively:

<CardGroup cols={2}>
  <Card title="Multi-cluster management" icon="network-wired">
    Control unlimited Kubernetes clusters from one intuitive interface, saving time on tool-switching and boosting productivity for DevOps teams.
  </Card>
  <Card title="AI-powered assistance" icon="robot">
    Leverage AI (OpenAI, Claude 4, Gemini, DeepSeek, OpenRouter, Ollama, Qwen, LMStudio) for automated troubleshooting, config optimization, and smart recommendations.
  </Card>
  <Card title="Single-binary deployment" icon="rocket">
    Install effortlessly as a lightweight binary on Mac, Windows, or Linux with no dependencies and zero configuration required.
  </Card>
  <Card title="Real-time monitoring" icon="chart-line">
    Get live views of clusters, pods, services, and metrics, enabling quick issue detection without manual queries.
  </Card>
</CardGroup>

## Key features

<AccordionGroup>
  <Accordion title="In-depth resource views">
    Dive into detailed manifests, logs, and configurations through an intuitive dashboard, making debugging a breeze for novices and pros alike.
  </Accordion>
  <Accordion title="Browser-based access">
    Access securely via any browser with optional HTTPS setup, perfect for remote teams managing on-premises or cloud clusters.
  </Accordion>
  <Accordion title="Advanced search & filter">
    Instantly locate namespaces, labels, images, nodes, and workloads with powerful search and filtering—streamlining navigation across large clusters.
  </Accordion>
  <Accordion title="Privacy by default">
    Maintain full control with zero cloud dependency, ensuring your cluster data stays local and secure by design.
  </Accordion>
  <Accordion title="One-click port forwarding">
    Instantly access in-cluster services on your local machine with secure, one-click port forwarding. No complex CLI commands or YAML edits required.
  </Accordion>
  <Accordion title="Live refresh">
    Experience seamless auto-updates for resources, eliminating manual refresh cycles and keeping your dashboard perpetually current.
  </Accordion>
  <Accordion title="Aggregated pod logs">
    Stream logs across pods and containers with advanced search and tail options—perfect for monitoring multi-replica applications with ease.
  </Accordion>
  <Accordion title="Clean resource management">
    Enjoy streamlined views for Deployments, Pods, Services, ConfigMaps, and more. Scale deployments, restart pods, perform rollout restarts, and apply manifests with a single click.
  </Accordion>
</AccordionGroup>
