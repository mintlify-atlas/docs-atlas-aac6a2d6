## List CronJobs

<api method="GET" endpoint="/api/v1/cronjobs" />

Retrieve a list of all cronjobs across namespaces or within a specific namespace.

### Query Parameters

<ParamField query="config" type="string">
  Configuration identifier for the Kubernetes context
</ParamField>

<ParamField query="cluster" type="string" required>
  Cluster name to query
</ParamField>

<ParamField query="namespace" type="string">
  Filter cronjobs by namespace. If not specified, returns cronjobs from all namespaces
</ParamField>

### Response

Returns an array of cronjob objects with their specifications and status.

<ResponseField name="uid" type="string">
  Unique identifier for the cronjob
</ResponseField>

<ResponseField name="namespace" type="string">
  Namespace where the cronjob is deployed
</ResponseField>

<ResponseField name="name" type="string">
  Name of the cronjob
</ResponseField>

<ResponseField name="spec" type="object">
  CronJob specification
  <ul>
    <li><code>schedule</code> (string): Cron schedule expression</li>
    <li><code>concurrencyPolicy</code> (string): How to handle concurrent executions (Allow, Forbid, Replace)</li>
    <li><code>suspend</code> (boolean): Whether the cron job is suspended</li>
    <li><code>successfulJobsHistoryLimit</code> (integer): Number of successful jobs to retain</li>
    <li><code>failedJobsHistoryLimit</code> (integer): Number of failed jobs to retain</li>
  </ul>
</ResponseField>

<ResponseField name="status" type="object">
  Current cronjob status
  <ul>
    <li><code>active</code> (integer): Number of currently running jobs</li>
    <li><code>lastScheduleTime</code> (string): Last time the job was scheduled</li>
    <li><code>lastSuccessfulTime</code> (string): Last time the job completed successfully</li>
  </ul>
</ResponseField>

<ResponseField name="age" type="string">
  CronJob creation timestamp
</ResponseField>

### Example Request

```bash
curl -X GET "https://your-kubewall-instance/api/v1/cronjobs?cluster=production&namespace=default" \
  -H "Content-Type: application/json"
```

### Example Response

```json
[
  {
    "uid": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "namespace": "default",
    "name": "backup-cronjob",
    "spec": {
      "schedule": "0 2 * * *",
      "concurrencyPolicy": "Forbid",
      "suspend": false,
      "successfulJobsHistoryLimit": 3,
      "failedJobsHistoryLimit": 1
    },
    "status": {
      "active": 0,
      "lastScheduleTime": "2024-03-03T02:00:00Z",
      "lastSuccessfulTime": "2024-03-03T02:00:00Z"
    },
    "age": "2024-03-01T10:15:30Z"
  }
]
```

## Delete CronJob

<api method="DELETE" endpoint="/api/v1/cronjobs" />

### Query Parameters

<ParamField query="config" type="string">
  Configuration identifier for the Kubernetes context
</ParamField>

<ParamField query="cluster" type="string" required>
  Cluster name
</ParamField>

<ParamField query="namespace" type="string" required>
  Namespace of the cronjob
</ParamField>

</ParamField>
