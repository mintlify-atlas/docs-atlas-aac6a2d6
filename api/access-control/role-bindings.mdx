## List Role Bindings

Retrieve a list of all role bindings in the cluster.

### Endpoint

```
GET /api/v1/rolebindings
```

### Query Parameters

<ParamField query="config" type="string" required>
  The kubeconfig identifier for the cluster connection
</ParamField>

<ParamField query="cluster" type="string" required>
  The cluster name to query
</ParamField>

<ParamField query="namespace" type="string">
  Filter role bindings by namespace. If not provided, returns role bindings from all namespaces.
</ParamField>

### Response

Returns an array of role binding objects.

<ResponseField name="uid" type="string">
  Unique identifier for the role binding
</ResponseField>

<ResponseField name="namespace" type="string">
  The namespace where the role binding is located
</ResponseField>

<ResponseField name="name" type="string">
  The name of the role binding
</ResponseField>

<ResponseField name="age" type="string">
  Creation timestamp of the role binding
</ResponseField>

<ResponseField name="subjects" type="object">
  Subject bindings for this role binding
  <Expandable title="subjects">
    <ResponseField name="bindings" type="array">
      Array of subject names bound to this role
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.kubewall.io/api/v1/rolebindings?config=default&cluster=production&namespace=default" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

### Example Response

```json
[
  {
    "uid": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "namespace": "default",
    "name": "read-pods",
    "age": "2024-01-15T10:30:00Z",
    "subjects": {
      "bindings": ["jane", "bob"]
    }
  }
]
```

## Delete Role Binding

Delete one or more role bindings.

### Endpoint

```
DELETE /api/v1/rolebindings
```

### Query Parameters

<ParamField query="config" type="string" required>
  The kubeconfig identifier for the cluster connection
</ParamField>

<ParamField query="cluster" type="string" required>
  The cluster name to query
</ParamField>

<ParamField query="namespace" type="string" required>
  The namespace where the role binding is located
</ParamField>

<ParamField query="name" type="string" required>
  The name of the role binding to delete. Can be a comma-separated list for multiple deletions.
</ParamField>

### Example Request

```bash
curl -X DELETE "https://api.kubewall.io/api/v1/rolebindings?config=default&cluster=production&namespace=default&name=read-pods" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```
