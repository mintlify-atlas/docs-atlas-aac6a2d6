## List Service Accounts

Retrieve a list of all service accounts in the cluster.

### Endpoint

```
GET /api/v1/serviceaccounts
```

### Query Parameters

<ParamField query="config" type="string" required>
  The kubeconfig identifier for the cluster connection
</ParamField>

<ParamField query="cluster" type="string" required>
  The cluster name to query
</ParamField>

<ParamField query="namespace" type="string">
  Filter service accounts by namespace. If not provided, returns service accounts from all namespaces.
</ParamField>

### Response

Returns an array of service account objects.

<ResponseField name="uid" type="string">
  Unique identifier for the service account
</ResponseField>

<ResponseField name="namespace" type="string">
  The namespace where the service account is located
</ResponseField>

<ResponseField name="name" type="string">
  The name of the service account
</ResponseField>

<ResponseField name="age" type="string">
  Creation timestamp of the service account
</ResponseField>

<ResponseField name="spec" type="object">
  Service account specification
  <Expandable title="spec">
    <ResponseField name="secrets" type="number">
      Number of secrets associated with this service account
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.kubewall.io/api/v1/serviceaccounts?config=default&cluster=production" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

### Example Response

```json
[
  {
    "uid": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "namespace": "default",
    "name": "default",
    "age": "2024-01-15T10:30:00Z",
    "spec": {
      "secrets": 1
    }
  }
]
```

## Delete Service Account

Delete one or more service accounts.

### Endpoint

```
DELETE /api/v1/serviceaccounts
```

### Query Parameters

<ParamField query="config" type="string" required>
  The kubeconfig identifier for the cluster connection
</ParamField>

<ParamField query="cluster" type="string" required>
  The cluster name to query
</ParamField>

<ParamField query="namespace" type="string" required>
  The namespace where the service account is located
</ParamField>

<ParamField query="name" type="string" required>
  The name of the service account to delete. Can be a comma-separated list for multiple deletions.
</ParamField>

### Example Request

```bash
curl -X DELETE "https://api.kubewall.io/api/v1/serviceaccounts?config=default&cluster=production&namespace=default&name=my-service-account" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```
